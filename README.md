🚀 MC-Tunnel v0.5 — Minecraft Reverse Proxy & Web Dashboard

Высокопроизводительный реверс-прокси на Rust для проброса Minecraft-серверов из домашней сети на VPS, с поддержкой веб-панели мониторинга в реальном времени.
🌟 Что нового в v0.5

    Web Dashboard API: Полная статистика сервера доступна через REST API.

    IP Tracking: Мониторинг подключенных игроков в реальном времени.

    Encapsulated Core: Переписанная архитектура на геттерах для стабильной работы под нагрузкой.

    Traffic Stats: Учет общего трафика и пиковых нагрузок.

🛠 Быстрый старт
1. Подготовка конфига

Скопируйте пример конфигурации и настройте его под свои нужды:

cp config.example.toml config.toml

2. Настройка config.toml

Отредактируйте файл, указав нужный режим (vps или client):
Ini, TOML

mode = "vps"             # Или "client" для домашнего ПК
game_port = 25565        # Порт, на котором игроки видят сервер (только для VPS)
tunnel_port = 15000      # Порт для связи между VPS и Клиентом
vps_address = "IP_ВАШЕГО_VPS:15000"
minecraft_address = "127.0.0.1:25565"
web_port = 3001          # Порт для веб-дашборда

3. Запуск
Bash

# Сборка и запуск
cargo run --release

📊 Web Dashboard

Если вы запустили сервер в режиме vps, дашборд будет доступен по адресу:
http://ваш-ip:3001

Доступные метрики:

    Статус туннеля (Online/Offline).

    Список активных IP игроков.

    Аптайм и пиковое количество подключений.

    Общий объем переданных данных.

🔧 Технологии

    Backend: Rust, Tokio (Async IO), Axum (Web API).

    Frontend: React, TypeScript, Tailwind CSS.
